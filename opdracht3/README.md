## Evaluatie Dagelijks Werk

Deze week krijg je de tijd om de vorige projecten te verbeteren, onder andere op basis van de peer reviews die je ontving. De evaluatie gebeurt op basis van de code die je aan het einde van deze week

De rubric hieronder toont hoe je zult geëvalueerd worden. Deze evaluatie kan je gebruiken om te zien op welke vlakken je je code nog moet verbeteren tegen het examenresultaat.
